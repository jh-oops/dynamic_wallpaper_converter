动态壁纸打包器

按规范打包动态壁纸：H.264 转码 / 抽帧 / 缩略图 / manifest，输出标准 ZIP。

1上传视频并填写信息

拖入 mp4 到这里，或点击选择
支持任意来源：非 H.264 / 分辨率不对会自动转码处理，铺满全屏
动态预览图为探索页推荐用，规范上限 ≤2M；复杂视频可能超限并告警。

2上传文件并识别分类/标签

拖入图片或视频（mp4 / jpg / png / webp 等）
引擎会“读图理解”——分析主色、亮度、饱和度、细节密度，结合文件名命中你的标签库；你可在下方修改。
识别依据（图像特征）
建议 3–5 个标签；点击标签即可选中/取消。

标签库管理

按“一级分组 → 二级条目”管理分类与标签。点击分组标题可折叠；修改后点“保存标签库”立即生效。

3上传图片/视频并选择目标尺寸

拖入图片或视频到这里，或点击选择
支持静态图片（JPG / PNG / WebP 等）与视频 MP4；仅做尺寸裁剪/缩放，不打包进动态壁纸 zip

1上传 Excel（含 名称 / 简介 / 定价 / 分类）

拖入 Excel（.xlsx）到这里，或点击选择
理论上初始会给你一份 Excel 已经包含：名称、简介、定价、分类
输出 Excel 会新增/填充两列：设计师id、标签。名称列（去扩展名）将与图片文件名完全对应。
